Brief about WP CLI

Brief about WP CLI

WP-CLI is a text-based tool that allows you to manage your WordPress site using simple commands instead of a graphical interface (GUI).

WP-CLI lets you perform a wide range of tasks like installing plugins, managing users, creating posts and pages, and more, all from the command line.

Example below is using the wp cli to list the users on this site

						
-bash-4.2$ wp plugin list
